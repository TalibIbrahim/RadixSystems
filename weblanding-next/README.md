This is a migrated Next.js scaffold of the Vite-based landing page.

How to run locally:

1. cd weblanding-next
2. npm install
3. npm run dev

Notes:

- The `Aurora` component uses `ogl` and is loaded client-side with `next/dynamic`.
- The contact form posts to `/api/contact` (currently logs on the server).
- Add environment variables and integrate an email provider for production.
